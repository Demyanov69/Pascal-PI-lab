# Pascal-PI-lab

## Лабораторные работы по программной инженерии

**Студент:** Паскал В. С.  
**Группа:** МО-32/1  
**Тема:** Разработка веб-системы для автоматизации образовательного процесса

### Описание проекта
Разработка программной системы для образовательной организации, реализующей дополнительные профессиональные программы и онлайн-курсы с применением дистанционных образовательных технологий.

### Функциональность
- Управление пользователями и доступом
- Управление учебным контентом
- Контроль знаний и обратная связь
- Аналитика и отчетность

### Технологический стек
- Backend: PHP, Laravel
- Frontend: JavaScript, React, Bootstrap
- База данных: MySQL
